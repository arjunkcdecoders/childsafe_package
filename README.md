# childsafe_package

A Flutter package to track and analyze keystrokes and swipe biometrics natively, assisting in persona metrics or child safety verification.

## Features

* **Keystroke Tracking:** Measure typing speed (Inter-Key Interval - IKI), backspace density, and typo ratios utilizing a built-in preloaded dictionary constraint.
* **Swipe Tracking:** Track screen swipes to compute physical layout metrics like geometric path distance, speed, and axis erraticness.

In your flutter project's `pubspec.yaml`, add the dependency:

  childsafe_package: ^1.0.0
